# React + Vite

This is a portfolio project built with React and Vite. It showcases personal work, skills, and projects in a responsive web UI.

## How to run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start dev server:

   ```bash
   npm run dev
   ```

3. Open in browser:

   - usually `http://localhost:5173`

## Build

```bash
npm run build
```

## Deploy

- For static-hosting platforms (Netlify, Vercel, GitHub Pages, Cloudflare Pages):
  - Build first: `npm run build`
  - Deploy contents of `dist/`
- Many personalization setups support a convenience deploy command:
  - `npm run deploy` (publishes to https://ajingopi.github.io/ajingopi-portfolio/)
- For Vercel, simply connect the repo and use:
  - build command: `npm run build`
  - output directory: `dist`
